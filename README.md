This is an ongoing effort to implement Prism in Rust.

# Build

Run `cargo build`.

The first build could take several mintues, mostly due to building RocksDB.

# Run Tests

Run `cargo test`. To see all the outputs, append `-- --nocapture`.

# See Docs

Run `cargo doc --open`.
